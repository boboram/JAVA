QUEUE 
- 먼저 들어온 데이터가 먼저 나온다는 FIFO(First In First Out) 개념을 가지고 있습니다.

링크드 큐
- 환형큐와 달리 동적으로 연결이 가능한 큐 구현 방식입니다.
- 배열이 아니기 때문에 isFull()함수는 필요하지 않습니다.
- 데이터가 추가되면 후단 노드뒤에 새로운 노드 연결, 후에 후단 노드를 추가된 노드로 변경합니다.
- 데이터 삭제시 전단 노드를 제거되는 노드의 다음연결 노드로 변경합니다.

C언어와 JAVA의 구현 차이점
- 구조체는 C언어에는 있지만 JAVA에는 존재하지 않습니다. 
    때문에 클래스를 선언하여 객체를 생성합니다.(인스턴스화) 또한 동적할당을 할 필요도 없습니다.
- JAVA에는 GC가 있기 때문에 메모리를 개발자가 직접 제거하지 않아도 됩니다.
- C언어엔 없는 boolean타입을 사용하여 값의 유무 판단을 더 간편하게 할 수 있습니다.
